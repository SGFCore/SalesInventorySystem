# Hướng dẫn cài đặt và sử dụng frontend

### Thông tin cơ bản

**Công nghệ sử dụng**: ElectronJS, ReactJS, TS, TailwindCSS, ShadCN/UI.

**Yêu cầu cài đặt các phần mềm sau vào máy**:

- Node.js 18+ và npm.

### Các lệnh

**(Các lệnh sau chạy ở thư mục `frontend/`)**

**Tải dependencies** (chỉ chạy ở lần đầu tiên):

```bash
npm install
```

**Mở môi trường development**:

```bash
npm run dev
```

**Tắt server**: Gửi abort `Ctrl` `C`.
